POC 2: Rock Paper Scissors

Poc 2 is een rock paper scissors game met computer en met 2 spelers. Deze heb ik volledig zelf geprogrammeerd. De source code kan je kijken op Fontys Gitlab onder naam van 'poc2'. Je kan hier ook de Demo bekijken.

Aanpak van Rock Paper Scissors met computer

Ik heb eerst deze poc gemaakt met teksten en daarna heb ik deze uitgebreid met plaatjes. Hier heb ik uitgezocht hoe ik een plaatje in een Array kan doen en ook hoe je elke plaatje in het Array een naam kan geven.

Ik heb ook een timebox gemaakt zodat de speler eerst kan nadenken. Dit heb ik gedaan door de methode "setTimeout" te gebruiken. Vervolgens heb ik gekeken hoe je een random plaatje uit een Array kan halen en dat te laten zien op de website. Als laatst heb ik gekeken naar hoe je de plaatjes en teksten weg kan halen (resetten) op de moment de speler opnieuw begin met de game.

Aanpak van Rock Paper Scissors met 2 spelers

Ik heb eerst twee buttons gemaakt en daarna heb ik gekeken hoe ik een button na het drukken kan deactiveren zodat de ene speler niet meerdere keren kan drukken, maar past wanneer de andere speler gespleed heeft dan opnieuw kan spelen. Hiervoor heb ik de tutorials gevolgd van JS HTML DOM op w3school.com .

Voor het beslissen van winnaar heb ik gebruik gemaakt van if else statement. Eerst had ik "==" gebruikt in de condition, maar de resultaat valt tegen mijn verwachting. Na heel veel gevecht met if else statementen, random en Array kwam ik er achter dat je "===" moet gebruiken, omdat "===" nauwkeuriger is dan "==".

Wat heb ik geleerd?

Ik heb geleerd:

  • hoe ik een plaatje in een Array kan doen en elke plaatje een naam kan geven.
  • hoe ik met een functie een ander functie kan oproepen.
  • hoe ik met een methode een ander functie kan oproepen.
  • hoe ik een random element uit een Array kan halen.
  • wat Document Object Model is.
  • hoe ik een button kan deactiveren of activeren.
  • hoe Javascript condition werkt.
  • wat het verschil is tussen "===" en "==".
