Portfolio (UX)

Voor de interactieve media heb ik de UX van mijn portfolio getest door bij een aantal mensen feedback te vragen. Ik stel vragen zoals “Hoe ervaar je de look & feel, wat kan beter, hoe vindt je van de kleuren, zijn de teksten comfortabel om te lezen en etc”. Daarnaast heb ik ook feedback van docenten verwerkt die ik tijdens portfolio review heb gekregen.

Review 1

Deze review bevat alle feedback die ik heb gekregen sinds het ontwerpen van dit portfolio. In deze versie van mijn portfolio heb ik wireframes en mockups gemaakt in Adobe XD en ik heb de inhoud ook erin verwerkt, maar er mist wel wat development van het portfolio.

Een vriendin ( een fotograaf )

Een vriendin ( een fotograaf ) Deze persoon vindt dat de achtergrond van de oude versie niet zo mooi, maar ze vindt de nieuwe versie veel mooier en gestructureerd. Als feedback geeft ze verder dat ze het lastig vind wanneer ze klaar is met één project lezen terug naar boven moet om een ander project te lezen.

Klasgenoten

Ik heb mijn portfolio laten testen op 4 klasgenoten. Ze vonden mijn portfolio er gestructureerd en mooi uitzien. Één van mijn klasgenoot gaf als opmerking dat ik de rondje in de achtergrond kan laten animeren.
