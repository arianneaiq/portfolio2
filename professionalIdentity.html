Semester 2

Ik ben een student met alleen voorkennis van Semester 1. Hier zal ik het over semester 2 hebben. Over een toekomstige baan moet ik nog nadenken, maar ik denk wel dat ik in de richting wil van front-end developer, waar ook design bij komt kijken.

Wat heb ik in dit semester geleerd?

In dit semester heb ik me vooral gefocust op media technieken en developement. Zo heb ik nieuwe Adobe software geleerd zoals Photoshop, Premiere Pro, After Effects en Indesign. Hiervoor had ik ook geen enkel kennis van JavaScript, maar nu kan ik al de basis JavaScript programmeren. Ook kan ik nu projecten programmeren met Three.js, lil-gui en Anime.js. Daarnaast heb ik in dit semester ook andere software geleerd zoals Blender en SparkAr. Tevens heb ik opgemerkt dat ik veel meer leer in groepsverband, want zo kan ik dan hulp vragen bij andere groepsleden als ik bijvoorbeeld vast zit met een developement probleem. Research pak ik nu ook heel anders op dan voorheen, sinds ik gebruik maakte van methodiek in onderzoek, merk ik dat ik kritischer werd omtrent keuzes maken.

Wat wil ik in de volgende semester leren?

In de volgende semester wil ik me verdiepen in JavaScript programmeren en in alle gebieden van ICT & Media Design zoals virtual reality. Ook wil ik mijn grafische technieken verbeteren.
