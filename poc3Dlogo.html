Sprint X: Interactive 3D Logo

Dit is mijn Sprint X eindproduct en ook mijn vierde PoC voor Advanced development: een interactive 3D logo. Je kunt hier over de onzerzoek lezen die ik gedaan heb. Mijn doel in Sprint X is om een 3D-model te maken van mijn eigen logo en dit te animeren met de kennis die ik heb geleerd van Advanced Development. De source code kan je kijken op Fontys Gitlab.

Aanpak

Hiernaast zie je een video hoe ik aan mijn eindproduct ben gekomen.

Voor het importeren van 3D model van blender in three.js heb ik gebruik gemaakt van GLTFLoader.
