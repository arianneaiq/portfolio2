Portfolio Code

Ik ben eerst begonnen met een portfolio design waar ik feedback op had gevraagd van zowel medestudenten als een docent. Nadat ik een idee had hoe mijn portfolio eruit zal zien, ben ik begonnen met het programmeren van de site. Ik heb hiervoor HTML, CSS en JavaScript gebruikt. Het hele project heb ik bijgehouden op een internal Gitlab repository van Fontys Hogeschool en dus kan de sourcecode bekeken worden.

Structuur

Ik heb voor de structuur van mijn portfolio HTML en grid van CSS gebruikt. Hier heb ik de code zo veel mogelijk leesbaar behouden door zo efficient en minimalistisch te werken.

Opmaak

Voor alle opmaak van mijn site heb ik mijn eigen CSS geschreven inclusief de grid systeem. Ik heb niet gebruikt gemaakt van bootstrap, omdat ik graag wil leren hoe ik zelf een responsieve site kan maken en hoe grid werkt.
