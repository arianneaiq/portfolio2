Advanced development: Donuts

Dit is mijn eerste PoC voor Advanced development: 3 rond draaiende donuts. Klik hier voor de demo. De source code kan je kijken op Fontys Gitlab.

Onderzoek

Ik heb ondezoek gedaan naar verschillende libraries voor het maken van 3D website. Voor de onderzoek kun je hier verder lezen.

Aanpak

Ik ben eerst aan de slag gegaan met de documentatie van Three.js. Vervolgens heb ik Three.Js geinstalleerd met cdn. Daarna heb ik de basis van een 3D website opgezet namelijk scene, camera, object en renderer.

In de documentatie van three.js heb ik gekeken naar verschillende geometries(objecten) die er bestaan. Ik heb gekozen voor eem TorusGeometry, omdat ik in gedachte had om een donuts ervan te maken. De foto hiernaast is een donuts texture die ik uit het internet gehaald heb en rondom de objects gezet heb. Ik heb er in totaal 3 donuts met 3 verschillende texture gemaakt. Ik heb ook de achtergrond van de scene veranderd.

Als laatst heb ik de 3 donuts laten draaien opzichzelf in verschillende richtingen. Ook heb de donuts in een group gezet zodat de linker en rechter donuts om de middelste donuts kan draaien. Ik heb gebruikt gemaakt van window.requestAnimationFrame() om bij elke frame dit functie opnieuw to roepen en zo krijg je een oneidige loop waardoor je de animatie kan zien.

Framerate probleem

Zoals je op de console kunt zien, wordt op elk frame het 'animateDonuts' aangeroepen. Test je deze code op een computer met een hoge framerate, dan verschijnt het 'animateDonuts' met een hogere frequentie. Het probleem is dat als je deze code test op een computer met een hoge framerate, de donuts sneller zal draaien, en als je test op een lagere framerate, zal de donuts langzamer draaien. Kortom, de animatie ziet er anders uit op elke device. De oplossing hiervoor kun je vinden bij de volgende PoC.

Wat heb ik geleerd?

Ik heb geleerd hoe three.js werkt en hoe ik de basis setup kunt doen voor een 3D website. Verder heb ik geleerd hoe je de objecten kunt animeren. Ook heb ik heel toevallig de framerate probleem ontdekt doordat ik het website zowel op mijn pc als op mijn MacBook Pro had uitgetest.
