POC 1: Random Small Poc's

Ik heb nog nooit eerder met Javascript geprogrammeerd. Om hiermee te starten heb ik wat tutorials gevolg en ben ik direct van start gegaan met wat kleine poc's. Source code: Fontys Gitlab.

Tutorials

Om Javascript te leren ben ik eerst van start gegaan met tutorials op w3school.com. Hier heb ik tutorials gevolgd van JS Home tot JS Loop while. Ik heb ook de test youself opdrachten gemaakt om te kijken of ik Javascript echt snapt.

Kleine poc's

Na het volgen van tutorials ben ik zelf van start gegaan met poc's te maken. Ik heb zelf uitgeprobeerd:

  • hoe ik een html element met JavaScript kan veranderen.
  • hoe ik met een button click een functie kan activeren.
  • hoe ik variables kan aanmaken en functie kan oproepen met een methode.
  • hoe ik een string tot een int kan maken
