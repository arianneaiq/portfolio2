Special Week: Anime.js

Voor de Special Week heb ik mijn logo geanimeerd met Anime.js. De source code kan je kijken op Fontys Gitlab.

Onderzoek

In de special week heb ik verschillende JavaScript library zoals Vue.Js, ReactJS en etc onderzocht. Ik heb gekekn naar welk framework het meest gewild is op het werkveld. Hier kwam ReactJS uit als nummer 1. ReactJS legt de focus bij UI om op een snellere manier een mooie webapplicaties te ontwikkelen. Dit is alleen niet echt wat ik wilde, want ik zie mezelf als een media student daarom wilde ik graag iets doen dat visueel mooi en leuk was. Ik had sinds eerste semester iets willen doen met animeren, maar had er nog nooit iets aangedaan. Met dit special week wil ik mijn kans nemen om me te verdiepen in JavaScript library. Toen kwamen ik uit bij: • Anime.js • Velocity.js • Mo.js Uiteindelijk heb ik gekozen voor Anime.js, omdat deze library het meest gebruikte is en het heeft een goeie documentatie om mee te starten.

Aanpak

Ik heb eerst de documentatie van Anime.js door genomen. Vervolgens ben ik zelf gaan experimenteren met mijn logo. Wegens gebrek aan kennis van svg, liep het animeren van mijn logo niet zo vlot. Hiervoor heb ik eerst een onderzoek gedaan in svg.

Doordat de text rondom mijn logo niet draaide als ik wilde, heb ik mijn logo in twee aparte svg's gedaan en vervolgens met CSS bij elkaar toegevoegd. Zo kreeg ik precies de resultaat die je hiernaast in de video kan zien. De animatie kun je ook zien op mijn homepagina.
